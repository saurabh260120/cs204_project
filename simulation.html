SIMULATION PROCEDURE


  1. Turn On the power switch.
  2. Give the input by turning on buttons.(button is On means input is 1 and if button is Off it means input is 0.)
  3. Select the faulty Location from the drop down menu.
  4. Observe the expected output and faulty output on the bulb at right most corner.
  5. Add the value to the truth table by clicking the add button.
  6. The truth table will contain the expected output and the Faulty output.
  7. Print the truth table by clicking the PRINT button.

  8. Click the "open simulator" below to open the simulator.
